## 摘要
学习型 PDE 求解器通常被训练为针对特定方程、边界条件和离散化方式的整体代理模型。这使得它们在物理机制发生变化时难以重用，并可能限制长时程外推下的稳定性。我们引入了类乐高算子网络（LegONet），这是一个组合式框架，通过在共享的边界自适应谱表示上定义的即插即用、保结构算子块来构建 PDE 求解器。LegONet 将边界处理与机制学习分离，通过构造方式满足边界条件。它还将机制学习与时间积分分离，使得预训练的算子块无需重新训练即可组装成新的求解器。我们还推导了一种有限时程误差分解方法，将算子块失配与分裂误差分开，并为长时程预测提供机制层面的诊断。在十个含时 PDE 中，LegONet 实现了准确的闭环外推，并在跨 PDE 重组和边界重新配置下表现出更高的稳定性。更广泛地说，这种模块化表述为从特定任务的神经求解器迈向科学计算的即插即用算子库提供了一条路径。

## Abstract
Learned PDE solvers are often trained as monolithic surrogates for a specific equation, boundary condition and discretization. This makes them difficult to reuse when mechanisms change and it can limit stability under long-horizon rollout. We introduce Lego-like Operator Network (LegONet), a compositional framework that builds PDE solvers from plug-and-play, structure-preserving operator blocks defined on shared boundary-adapted spectral representations. LegONet separates boundary handling from mechanism learning, satisfying boundary conditions by construction. It also separates mechanism learning from time integration, enabling pretrained blocks to be assembled into new solvers without retraining. We also derive a finite-horizon error decomposition that separates block mismatch from splitting error and provides mechanism-level diagnostics for long-horizon predictions. Across ten time-dependent PDEs, LegONet delivers accurate closed-loop rollouts with improved stability under cross-PDE recombination and boundary reconfiguration. More broadly, this modular formulation suggests a path from task-specific neural solvers towards plug-and-play operator libraries for scientific computing.

### 1. 核心问题与整体含义（研究动机和背景）
传统的神经偏微分方程（PDE）求解器（如 FNO、DeepONet 或 PINNs）通常被设计为“单体式”（Monolithic）代理模型。这意味着它们针对特定的方程、边界条件（BC）和离散化方案进行训练，一旦物理机制发生变化（例如增加一个扩散项或改变边界条件），模型往往需要重新训练。此外，这些模型在长时程预测（Long-horizon rollout）中容易出现不稳定性。

**LegONet 的核心动机**是实现 PDE 求解器的“模块化”和“可重用性”。研究者希望像搭乐高积木一样，通过预训练一系列具有物理保结构特性的算子块（如扩散块、平流块、反应块），在无需重新训练的情况下，通过灵活组合这些块来构建复杂系统的求解器。

### 2. 方法论
LegONet 的核心思想是**解耦**：将边界处理与机制学习分离，将机制学习与时间积分分离。

*   **核心思想**：将 PDE 分解为多个物理机制之和（$u_t = \sum L_i u$），并在共享的系数空间（Coefficient Space）中表示这些机制。
*   **关键技术细节**：
    *   **边界自适应基座（Baseplate）**：利用谱方法（如 Fourier、Cosine、Shen-Legendre 基）构建满足特定边界条件的试验空间。通过“提升法”（Lifting）处理非齐次边界，使所有算子块都在同一组系数 $a(t)$ 上运行。
    *   **保结构算子块**：
        *   **E-blocks（耗散型）**：基于梯度流形式 $-\mathbf{G}\nabla E$ 参数化，保证能量耗散。
        *   **H-blocks（保守型）**：基于哈密顿形式 $\mathbf{J}\nabla H$ 参数化，保证能量守恒。
        *   **R-blocks（残差型）**：处理不符合上述形式的强制项或约束。
    *   **算子匹配训练（Operator Matching）**：算子块在离线状态下通过“瞬时匹配”进行预训练，即学习系数空间的向量场映射，而非拟合完整的轨迹。
    *   **组合推理**：在部署时，使用**对称 Strang 分裂法（Strang Splitting）**将选定的预训练块组合起来进行时间步进。

### 6. 主要结论与发现
*   **即插即用性**：预训练的算子块可以在不同方程间无缝迁移，无需针对新方程重新训练。
*   **长时稳定性**：相比于 FNO 和 DeepONet，LegONet 在长时程闭环预测中表现出极高的稳定性，尤其是在处理湍流和刚性（Stiff）方程时。

### 7. 优点（亮点）
*   **模块化架构**：首次实现了真正意义上的神经算子“乐高化”，极大地提高了模型的复用率。
*   **物理一致性**：通过硬编码耗散和保守结构，确保了数值模拟的物理合理性。
*   **高效率**：算子匹配训练不需要生成昂贵的长轨迹数据，只需瞬时快照即可。
*   **灵活性**：支持在推理阶段动态调整方程参数（如扩散系数）或重新配置边界条件。

### 8. 不足与局限
*   **基座依赖性**：算子块目前与特定的谱基底（Baseplate）绑定，跨不同基底（如从 Fourier 切换到多项式基）的直接迁移仍需进一步研究。
*   **几何限制**：目前主要针对规则几何区域（矩形、立方体），对于复杂不规则几何的适应性有待扩展。
*   **算子库完备性**：虽然支持组合，但如果遇到全新的非线性机制，仍需预训练新的算子块。
*   **分裂误差**：对于强非对易（Non-commuting）的多物理场耦合，Strang 分裂带来的数值误差可能会累积，需要更小的时间步长。
